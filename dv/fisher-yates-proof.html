Visual Proof of Fisher-Yates Shuffle

I have found great visualization of the fisher-yates shuffling algorithm, but have yet to find a great visualization of the proof that the fisher-yates shuffle is random.

What do we mean by random? One way to proove a shuffle is (uniformly) random is to show that every element has equal probability of being shuffled to every position, including the original position.

Permutations

Another way to prove that a shuffle is random is show that every permutation is equally possible as a result of the shuffle.

A permutation is reordering of a list. For example, [2,1,3,4,0] is a permutation of [0,1,2,3,4].

Represents permuting [0,1,2,3,4] to [2,1,3,4,0].

Permutation Matrix

Every permutation can be visualized as a special matrix called a permutation matrix. The columns represent the original position of an element while the rows represent the permuted position of an element. Only a single 1 entry per row or column is allowed, which signifies that the element at position column is being moved to position row.

Colored squares represent 1. Empty squares represent 0.

Algorithm

The fisher-yates algorithm shuffles by swapping the last element, en, with an element ei at a random position i, then recursing on all elements from e0 to en-1. In software we call those remaining elements the sublist from 0 to n-1.
